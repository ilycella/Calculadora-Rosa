🎀 Calculadora Rosa

Projeto de uma calculadora web funcional e com design estilizado, desenvolvido como atividade prática para a disciplina de **3DAW** da faculdade.

## ⚙️ Funcionalidades

* Realização de operações matemáticas básicas (soma, subtração, multiplicação e divisão).
* Limpeza de tela e reset de cálculos.
* Interface temática com botões estilizados e feedback visual de clique.

## 🛠️ Tecnologias Utilizadas

* **HTML5** (Estruturação semântica)
* **CSS3** (Estilização, layout e paleta de cores)
* **JavaScript** (Manipulação do DOM e lógica das operações matemáticas)
